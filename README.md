# template-poly
这是一个可以使用的稳定的版本，一个 release 版本。

OI 中的多项式模板，使用 `std::vector<modint<998244353>>` 实现。目前已实现：
- modint
- 单点求值
- NTT
- 多项式加法、减法、数乘
- 多项式乘法
- 多项式乘法逆
- 分式 $f(x)/g(x)$ 的第 $n$ 项
- 常系数齐次线性递推
- 求最短递推式的 BM 算法
- $O(n^2)$ 的拉格朗日插值
- 多项式对数函数
- 多项式指数函数
- 多项式快速幂
- 多项式除法、取模
- 多项式开根
